# rg-fzf-ctrl-y

Скрипт включает интерактивный поиск и при этом позволяет менять режим rg/fzf. Работает как CTRL+T, возвращает путь до файла

Для установки необходимо:

1) rg-fzf-ctrl-y переместить в .local/bin
2) содержимой dotfile переместить в .bashrc (не обязательно так, можно через source загружать его содержимое)
3) bind -x '"\C-y":__fzf_rg_widget' поменять на своё
